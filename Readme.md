# She Can Foundation Website

A responsive website created as part of the internship selection process for She Can Foundation.

## Features

* Responsive design for desktop and mobile devices
* Hero section with call-to-action
* About Us section
* Contact and social media links
* Modern UI using HTML and CSS

## Technologies Used

* HTML5
* CSS3
* Flexbox
* Grid
* Media Queries

## How to Run

Open `main.html` in any modern web browser.
